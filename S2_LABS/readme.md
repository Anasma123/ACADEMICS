## 🛠️ Quick Start

### Java on PuTTY
```bash
javac Program.java
java Program
````

### Networking & System Admin on Git Bash

```bash
./admin_script.sh
```

### ADBMS on MySQL CLI

```sql
SOURCE db_script.sql;
```

### MongoDB on Mongo Shell

```bash
test > use dbs
```
